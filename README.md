# AxionHub
🔴 AxionHUB — Automated Software Manager
> **AxionHUB** — це швидкий та зручний менеджер софту, створений для автоматичного завантаження та розгортання необхідних програм на свіжу систему в один клік.
> **швидкий**, **зручний** та **кросплатформений** інструмент для автоматичного завантаження та розгортання необхідного софту на **Windows** та **Linux**. Забудьте про ручний пошук інсталяторів — обирайте потрібні додатки та встановлюйте їх у кілька кліків!

---

## 🛠️ Як розроблялася програма?
Ідея проєкту полягала в тому, щоб прибрати нудну рутину вручну шукати й скачувати десятки інсталяторів через браузер.

1. **Ядро програми (Python)**
GUI (Інтерфейс): Написано на Python за допомогою сучасних графічних елементів (Tkinter / CustomTkinter) у фірмовому стилі Red Edition (темний вугільний фон із неоново-червоними акцентами).

Логіка завантаження: Програма автоматично звертається виключно до офіційних серверів розробників ПО, що гарантує 100% безпеку від вірусів та стороннього "сміття".

2. **Компіляція (Nuitka)**
Щоб програма працювала швидко й не вимагала встановленого Python на ПК користувача, код було скомпільовано за допомогою комбілятора Nuitka:

Усі залежності та ресурси згруповано в один автономний .exe файл.

Приховано вікно консолі для чистішого вигляду.

Вбудовано фірмову іконку red_icon.ico.

3. **Веб-сайт та дистрибуція**
Паралельно було створено офіційний веб-сайт-лендинг, оптимізований для слабких ПК (без важких скриптів), який підключено до GitHub Releases для прямого завантаження готового .exe.

## ⚡ Запуск
хоча програма сама і без залежностей пайтону і самого пайтону може запуститись
але, все ж якщо є проблеми то раджу вам завантажити файл requirements.txt 
і покласти його в свою /User/ папку потім cmd.exe вписать команду: pip install -r requirements.txt

## ⚡ Запуск з вихідного коду
Якщо ви хочете протестувати або модифікувати вихідний код проєкту:

**Встановіть необхідні залежності:**
pip install customtkinter
pip install requests

**Запустіть головний файл:**
python AxionHUB.pyw (качаєте якщо що в релізах "Source code(.zip)" або "Source code(tar.gz)")
